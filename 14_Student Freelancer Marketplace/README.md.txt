# 04_Student_Freelancer_Marketplace

## 1. Project Title
**Student Freelancer Marketplace** - A platform where students can offer skills and find freelance opportunities.

## 2. Project Description
He ek web platform aahe jithe vidyarthi aapan shikलेले skills (jas ke coding, graphic design, content writing) vaprun freelance kaam shodhu shaktat. Ya mule student-to-student kinva student-to-client networking madat hote.

## 6. Prerequisites and Installation
Hya project sathi tumchya PC var **Node.js** install asne garjeche aahe.

**Steps to Run:**
1. Folder open kara aani terminal/cmd madhye ja.
2. `npm install` type kara (Sagle dependencies install hotil).
3. `npm start` type kara.
4. Browser madhye `localhost:3000` var project suru hoil.
