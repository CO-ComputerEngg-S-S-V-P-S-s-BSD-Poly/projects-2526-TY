# 🚀 Blockchain for Student Record System - Group 12

**Project Topic**: Blockchain for Student Record Verification

## Prerequisites
- Node.js v14+
- MongoDB local (`mongodb`)

## Backend (`BackEnd/`)
```bash
cd BlockChain_For_Student_Record_System/BackEnd
npm install
node app.js  # http://localhost:8000
```

## Frontend (`FrontEnd/`)
```bash
cd BlockChain_For_Student_Record_System/FrontEnd
npm install
npm start  # http://localhost:3000
```

## Troubleshooting
- MongoDB: Run `mongodb`
- Ports: Kill processes or change PORT
